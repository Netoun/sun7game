## sun7game

A game of burger.

## Install and run

Run next commands in your terminal:

| Command        | Description                                                                                 |
| -------------- | ------------------------------------------------------------------------------------------- |
| `yarn install` | Install dependencies and launch browser with examples.                                      |
| `yarnyz start`   | Launch browser to show the examples. <br> Press `Ctrl + c` to kill **http-server** process. |
